# ECZ Exam Quiz Generator

This is a Flask-based application designed to generate quizzes based on ECZ exam questions. It allows users to take quizzes and track their progress.

## Setup Instructions
1. Clone this repository: `git clone https://github.com/hasiciimbwei-ops/wertinu`
2. Navigate to the project directory: `cd wertinu`
3. Install the required dependencies: `pip install -r requirements.txt`
4. Run the application: `python app.py`

## Features
- Flask web framework
- CORS support for API requests
- Environment variable management through dotenv
- Dynamic quiz generation from a database of questions.
